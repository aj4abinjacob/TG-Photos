    Search for BotFather in telegram

    Send "/start" and then "/newbot" to BotFather

    Then choose a name and username for your bot

    You will get a message with bot token, please copy your bot token

    Now search for your bot in telegram with the username and send "\start" and a "Hi"

    Go to https://api.telegram.org/bot<your-bot-token>/getUpdates

    Find "id" under "from" make sure the username is yours

    if id has not appeared, send a message to your bot again and refresh the webpage

    Copy and paste both the token and the chatid in the task in [tasker](https://play.google.com/store/apps/details?id=net.dinglisch.android.taskerm&hl=en_IN&gl=US) after importing the task
    
    If you encounter file not found error select "continue after error" inside the first task
